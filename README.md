# Introduction to Research

This repository contains the R code for the Introduction to Research project:
"From Raw Data to Structure: Dataset Curation and Clustering Analysis".

Contents:
- Script_Preprocessing.R: Data cleaning and preprocessing.
- Bootstrap_PAM_Clustering.Rmd: Clustering analysis and evaluation

Dataset:
European Social Survey (ESS) Round 10
https://ess.sikt.no/en/datafile/f37d014a-6958-42d4-b03b-17c29e481d3d
